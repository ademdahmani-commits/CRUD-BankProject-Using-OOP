# Bank Project (C++ OOP - Console Application)

A simple banking system built using C++ and Object-Oriented Programming concepts.
The project simulates basic banking operations through a console-based interface.

## Features

* Client Management

  * Show Client List
  * Add new client
  * Delete client
  * Update client information
  * Find client
 
* User Management

  * List users
  * Add new user
  * Delete user
  * Update user information
  * Find user

* Transactions

  * Deposit
  * Withdraw
  * View total balances

* Menu-driven navigation system

* Separation of concerns using multiple classes

## Project Structure

The project is divided into multiple header files, each responsible for a specific part of the system:

* `clsUsersListScreen.h` → Display and manage users list
* `clsWithdrawScreen.h` → Handle withdraw operations
* `clsUtil.h` → Utility functions used across the project
* Other classes handle business logic and data processing

## Technologies Used

* C++
* OOP principles:

  * Encapsulation
  * Abstraction
  * Separation of responsibilities...

## How to Run

1. Open the project in Visual Studio
2. Build the solution (Release x64 recommended)
3. Run the `.exe` file

## Notes

* This project is console-based (CMD)
* Data handling is simplified file system (no database integration)
* Intended for learning and practicing OOP design
